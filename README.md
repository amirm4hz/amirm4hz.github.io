# Amir Mahdian — Portfolio

> Personal portfolio website built from scratch with vanilla HTML, CSS, and JavaScript.
> Live at: [amirm4hz.github.io](https://amirm4hz.github.io)

## Tech Stack

- **HTML / CSS / Vanilla JS** — no frameworks, no build step
- **GSAP + ScrollTrigger** — all animations and scroll effects
- **Vanilla Tilt.js** — 3D card hover effect on projects
- **Formspree** — contact form email delivery
- **Google Fonts** — Syne (display) + Inter (body)
- **GitHub Pages** — free static hosting

## Sections

| Section | Description |
|---|---|
| Hero | Animated name reveal + typewriter cycling through domains |
| About | Bio, fun facts, avatar |
| Projects | 3 featured projects with GitHub links |
| Skills | Grouped tech stack badges |
| Contact | Formspree-wired form + social links |

## Local Development

No build step needed — just open the file:

```bash
git clone https://github.com/amirm4hz/amirm4hz.github.io.git
cd amirm4hz.github.io
open index.html   # Mac
# or drag index.html into any browser
```

## Deployment

Deployed automatically via GitHub Pages on every push to `main`.
